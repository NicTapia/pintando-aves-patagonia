## Logo

| Nombre de archivo  | Descripción                                       |
|--------------------|---------------------------------------------------|
| `logo-real.png`    | Logo real (PNG con fondo transparente, ideal)     |
| `logo-real.jpg`    | Logo alternativo en JPG (si no tenés PNG)         |

> Si usás JPG para el logo, cambiar la línea en index.html:
> `src="assets/logo-real.png"` → `src="assets/logo-real.jpg"`

## Fondo del Hero (opcional)

| Nombre de archivo    | Descripción                                        |
|----------------------|----------------------------------------------------|
| `video-lluvia.mp4`   | Video de lluvia/naturaleza para el tema oscuro     |
| `paisaje-sol.jpg`    | Foto de lago/bosque patagónico para el tema claro  |

> Estos dos son **opcionales**: si no están, el hero muestra un degradado de color.
> Para el video recomendamos: resolución 1920×1080, duración 10–30s, formato H.264.

## Formatos recomendados para las fotos de productos

- **JPG** para fotos de cerámica/tazas/teteras (fondo claro o neutro)
- Tamaño mínimo recomendado: **800×800 px**
- Relación de aspecto: **1:1** (cuadrada) para mejor visualización en las tarjetas
